OpenRP Documentation

Introduction to OpenRP

Hey there! Thanks a lot for downloading OpenRP! This intro is a basic guide into getting into OpenRP and using it. For more detailed information, check out the content below and use the sidebar links to the left to quickly navigate to a certain section! Note that commands are case insensitive so it doesn't matter whether you capitalize certain letters or not like they may be in the below lists.

To start OpenRP, run the play_OpenRP.bat file in your GameData folder.

Account Commands

Back to the top of the page

accountInfo - Display the account information of yourself or another player.

Usage: /acccountInfo - Display your own account info; /accountInfo <username> - Display the account info of another player

accountName - Displays the account name/username of another player who is current online by using their player name.

Usage: /accountName <playerName>

editAccount - Edit options/settings of your account.

Usage: /editAccount username <newUsername>; /editAccount password <newPassword>

login - Login to your account.

Usage: /login <username> <password>

logout - Logout of your account.

Usage: /logout

register - Register a new account.

Usage: /register <username> <password>

amAllChat - Show all chat on the server despite distance.

Usage: /amAllChat - Turn regular all chat on. You won't see your messages repeated with regular all chat on; /amAllChat complete - Turn all chat complete on. You will see your messages repeated in this mode


amAnnounce - Announce/broadcast a message to the entire server or a specific player. The message is shown in the center of their screen. Use \n for new lines.

Usage: /amAnnounce <name/clientID> <message>

amAudio - Play a sound file that the entire server hears. Play a music file that plays as the map's music (thus it automatically repeats),or turn off the map's music.

Usage: /amAudio <filePath> - Play a sound or music file. Sound files must be in the sound/ folder, music files must be in the music/ folder; /amAudio none - Turn off the map music

amBan - Ban a player from the server permanently.

Usage: /amBan <name/clientID>

amCheckStats - View the force powers/levels a player has.

Usage: /amCheckStats <name/clientID>


amCreateFaction - Create a faction. Thisautomatically makes yourself the leader of it. Only admins can use this command.

Usage: /amCreateFaction <factionName>


amDisguise - Disguise/undisguise your status as an admin to regular players in the admin list.

Usage: /amDisguise


amEffect - Place an effect on the location you're standing on. Prints the effect ID so it can be easily removed later.

Usage: /amEffect <effectName>

amEmp/emPower - Give yourself or another player every force power at level 3.

Usage: /amEmp (or /amEmpower) - Empower yourself; /amEmp <name/clientID> (or /amEmp) - Empower another player

amFactionGenCredits - Generate credits that are generated in a faction's bank.

Usage: /amFactionGenCredits <factionID> <amount>

amForceTeam - Force someone to a different team like spectator.

Usage: /amForceTeam <name/clientID> <newTeam> - Valid options for the new team are: spectator, spec, s, for the spectator team, and free, f, for the free team.


amGenCredits - Generate credits for a specific character.

Usage: /amGenCredits <characterName> <amount>

amGhost - Turn yourself invisible to all other players (does the same thing /amInvisible does).

Usage: /amGhost

amGiveAdmin - Give admin status to a player. Their admin status will remain even after logging out, but can be revoked with /amRemoveAdmin.

Usage: /amGiveAdmin <username> <adminLevel>

 

amGiveGun - Give a gun to a character.

Usage: /amGiveGun <characterName> <gunName>

Usage: /amGiveSkillPoints <characterName> <amount>

amInvisible - Turn yourself invisible to all other players.

Usage: /amInvisible


amKick - Kick a player from the server.

Usage: /amKick <name/clientID>


amListEnts - List the name of entities you have spawned along with their IDs.

Usage: /amListEnts


amListWarnings - List the amount of warnings each player has out of the amount of warnings required for a player to be automatically kicked from the server.

Usage: /amListWarnings


amLockSaberSetting - Lock a character's saber in training mode so it only does 1 damage. The lock will remain even when the player logs off. The training setting can be unlocked by using the command a second time on the character.

Usage: /amLockSaberSetting <characterName>


amMap - Change the server's map.

Usage: /amMap <mapName>


amMerc - Give a player every weapon with ammo.

Usage: /amMerc - Merc yourself; /amMerc <name/clientID> - Merc another player


amOrigin - Prints your or another player's XYZ coordinates.

Usage: /amOrigin - Prints your own coordinates; /amOrigin <name/clientID> - Prints another player's coordinates

amRemoveAdmin - Remove a player's admin status.

Usage: /amRemoveAdmin <username>

amRemoveEnt - Remove an entity by using its ID, even if you didn't spawn it. Find an entity's ID by using /amListEnts.

Usage: /amRemoveEnt <entID>


amRename - Change a player's display name to a new one.

Usage: /amRename <currentName/clientID> <newName>

amSetFaction - Set a character's faction if they're not currently in one, or remove them from their faction if they are currently in one.

Usage: /amSetFaction <characterName> <factionID> - Set a player's faction if they're not currently in one; /amSetFaction <characterName> none - Remove a player from their faction if they're currently in one.


amShakeScreen - Cause every player's screen to shake with a specified intensity for a specified amount of time in seconds.

Usage: /amShakeScreen <intensity> <length> - With intensity, 1 is a low amount, higher values are high amounts. Length is in seconds


amSilence - Silence a player so they cannot use chat.

Usage: /amSilence <name/clientID>


amSleep - Put a player to sleep so they are unable to move.

Usage: /amSleep <name/ClientID>


amSpawnEnt Spawn an entity from the entity list where you're looking.

Usage: /amSpawnEnt <entName>

amStatus - View a list of all players on the server along with their clientIDs and IP addresses.

Usage: /amStatus


amTele - Teleport one player to another, you to another player, to a location you marked, or to XYZ coordinates.

Usage: /amTele - Teleport to a location you marked with /amTeleMark; /amTele <name/clientID> - Teleports you to the specified player; /amTele <name/clientID> <name/clientID> - Teleport the first player specified to the second; /amTele <X> <Y> <Z> - Teleport yourself to specific XYZ coordinates; /amTele <name/clientID> <X> <Y> <Z> - Teleport a player to specific XYZ coordinates


amTeleMark - Marks your current XYZ coordinates so you can teleport to them with /amTele.

Usage: /amTeleMark


amTimer - Currently unused or disabled.

Usage:


amToBlack - Make all players screens either fade into blackness or immediately turn black. Fading in takes 10 seconds. Use the command a second time with the same parameter (fade/nofade) as you originally supplied to make players' screens normal again.

Usage: /amToBlack fade - Make all players' screens fade into blackness over a period of 10 seconds; /amToBlack nofade - Make all players' screens immediately turn black

amUnsilence - Unsilence a player who was silenced so they can chat again.

Usage: /amUnsilence <name/clientID>


amUnsleep - Unsleep a player who was slept so they can move again.

Usage: /amUnsleep <name/clientID>

amWarn - Give a player a warning. After a certain amount of warnings (set in server.cfg, default is 3), the player is automatically kicked from the server.

Usage: /amWarn <name/clientID>


amWeather - Change the weather on the server. Specify "none" as the weather effect to clear the weather.

Usage: /amWeather <weatherEffect>

amWeatherPlus - Add additional weather effects so multiple weather effects can be used at the same time. Specify "none" as the weather effect to clear the weather.

Usage: /amWeatherPlus <weatherEffect>

Character Commands

Back to the top of the page

bounty - View the bounty board, add bounties, and remove bounties if you're an admin

Usage: /bounty - View the bounty board; /bounty add <characterName> <reward> <0(dead)/1(alive)/2(dead or alive)> - Add a character to the bounty board, along with a specified reward for the person who completes the bounty as well as a number indictating the target is wanted dead, alive, dead or alive; /bounty remove <bountyID> - If you're an admin, remove a certain bounty from the board. Credits currently will need to be given manually to the bounty hunter who completes a bounty.

characterInfo - View information about one of your characters or another player's characters.

Usage: /characterInfo - View your own character info; /characterInfo <characterName> - View another character's info. It could be another one of your characters' info, or another player's character's.

character - Select a character of yours to use.

Usage: /character <characterName>

characterName - View the currently selected character of a player by providing their current display name.

Usage: /characterName <displayName>

checkInventory - Currently unused or disabled.

Usage:

createCharacter - Create a new character.

Usage: /createCharacter <name> <forceSensitive> - Force sensitive can be yes or true, no or false. Character names cannot have spaces.

editCharacter - Edit certain aspects/settings for your currently selected character.

Usage:/editCharacter name <newCharacterName> - Edit your character's name. Character names cannot have spaces; /editCharacter modelScale <value> - Value must from 65 - 140.

faction - Join a faction that is not invite only.

Usage:/faction <factionID> - Use /listFactions to find the factionID.

factionDeposit - Deposit some of your credits into your faction's bank.

Usage:/factionDeposit <amount>

factionInfo - View information about your faction.

Usage:/factionInfo - View your faction's name, ID, leaders, bank balance, and your rank; /factionInfo roster - View all of the members of your faction along with their ranks.

factionInvite - Invite a character to your faction if you are a leader of it.

Usage:/factionInvite <characterName>

factionLeave - Leave your faction.

Usage:/factionLeave

factionWithdraw - Withdraw credits from your faction bank. Only faction leaders can use this command.

Usage:/factionWithdraw <amount>

force - Speak to another person using the force if your character is force sensitive.

Usage:/force <name/clientID> <message>

frequency - Change your radio's frequency.

Usage:/radio <frequency> - Frequencies must be from 0 - 100

giveCredits - Give some of your credits to another character.

Usage:/giveCredits <characterName> <amount>

inventory - Check your inventory, equip weapons, and manipulate weapons.

Usage:/inventory - View a list of weapons in your inventory; /inventory use <weaponName> - Equip a weapon that is in your inventory

listFactions - View a list of factions, along with their IDs.

Usage:/listFactions

listCharacters - View a list of your characters on your account, along with their charIDs.

Usage:/listCharacters

radio - Talk over your radio to others on the same frequency.

Usage:/radio - Turn on or off your radio; /radio <message> - Talk over your radio

setFactionRank - Set a character's rank in their faction. This command can only be used by one of their faction's leaders or an admin.

Usage:/setFactionRank <characterName> <rankName>

shop - View the shop and buy weapons.

Usage:/shop - View a list of weapons in the shop along with their cost and level requirement; /shop buy <weaponName> - Buy a certain weapon if you have enough credits; /shop examine <weaponName> - View a description of the weapon

Chat Commands

Back to the top of the page

admin - Only usable if you're an admin. Chat with other admins. Only admins can send and see this chat.

Usage: /admin <text>

it - Perform an environmental action. Example: The tree fell.

Usage: /it <text>

looc - Local/nearby out of character chat.

Usage: /looc <text>

me - Have your character perform an action.

Usage: /me <text>

ooc - Serverwide out of character chat.

Usage: /ooc <text>


report - Report something to online admins. Only the reporter and the online admins will be able to see this.

Usage: /report <text>


whisper - Say something that can only be heard in a very small area

Usage: /whisper <text>


yell - Say something that can be heard in a very large area

Usage: /yell <text>

listAdmins - Displays a list of admins who are currently not disguising the fact that they are admins along with their admin level. If you are an admin and use this command, you will see a list of all admins and their levels, even if they are disguised.

Usage: /listAdmins

roll - You roll a random number between 1 and 100 which is displayed in chat.

Usage: /roll


setTraining - Either sets your lightsaber into training mode so it only does 1 damage, or if an admin uses this on someone, it sets their saber to training mode.

Usage: /setTraining - Changes your own lightsaber's mode to training mode. (Use the command a second time to change it back to the normal mode); /setTraining <playerName> - If you're an admin, this changes the specified player's saber mode so it's set to training mode. (Use the command a second time to change it back to the normal mode)


setStun - Sets your lightsaber to stun mode. It stuns the person who is hit. They are stunned for 10 seconds, are on the ground, and are unable to move for that duration of time.

Usage: /setStun TODO - Changes your own lightsaber's mode to training mode. (Use the command a second time to change it back to the normal mode); /setTraining <playerName> - If you're an admin, this changes the specified player's saber mode so it's set to training mode. (Use the command a second time to change it back to the normal mode)


toggleChat - Changes what type(mode) of chat team chat acts as. Your can either cycle through the various chat modes, or choose a specific mode.

Usage: /toggleChat - Everytime this is used, it cycles to the next chat mode. The order of the cycle is: OOC(the default mode), LOOC, Yell, Whisper, Me, It, Admin (only if you're an admin); /toggleChat <mode> - Sets your chat mode to a specific mode. Valid modes are: OOC(the default mode), LOOC, Yell, Whisper, Me, It, Admin (only if you're an admin). If you choose an invalid mode, the mode will automatically be set to OOC
